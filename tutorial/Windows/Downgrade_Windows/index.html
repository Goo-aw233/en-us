Downgrade Windows

Created2023-02-01|Updated2023-02-05|tutorialWindows
|Word count:165|Reading time:1min|Post View:

Setup could not keep files to downgrade the system before replacing setuphost.exe

Using UltraISO, replace the setuphost.exe file in the sources folder of the new version of the ISO with the old one and save it

If UltraISO could not save the file
(Sorry ~ there aren’t any screenshots here)
Extract all the contents of the old ISO, click on “Boot” at the top of UltraISO, and click on “Save boot file…” and save it to a random location

Copy the setuphost.exe file from the sources folder in the new version of the ISO and replace it with the old one

Create a new ISO with UltraISO, copy all the files from the old ISO, click “Start” on UltraISO, and select “Load boot file…” and select the boot file you just saved, and finally click Save

Once the replacement is complete, you can keep the files to downgrade Windows


Finally, you can use the downgraded Windows normally(However, there may be some strange bugs)
